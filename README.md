# Slatkisasa Tap

## How do I install these formulae?

`brew install slatkisasa/tap/<formula>`

Or `brew tap slatkisasa/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "slatkisasa/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
